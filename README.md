# TPs_Genie_Logicielle
# TP Architecture : Du Monolithe au Distribué

## 📌 Description
3 travaux pratiques progressifs sur l'évolution architecturale :
1. **Monolithe Spring Boot** (CRUD produits)
2. **Modularisation** (DTO, MapStruct, modules métier)
3. **Architecture distribuée** (Node.js, Nginx, MongoDB, Redis, RabbitMQ)

## 🛠 Technologies
`Spring Boot` `JPA` `PostgreSQL` `MapStruct` `Node.js` `Express`  
`Docker` `Nginx` `MongoDB` `Redis` `RabbitMQ` `Postman`

## 📂 Structure
- `tp1/` - API monolithique simple
- `tp2/` - Version modulaire + DTOs
- `tp3/` - Système distribué (6 services)

## 🚀 Objectifs
- Comprendre l'architecture en couches
- Préparer la migration microservices
- Observer load balancing, cache, bus de messages
